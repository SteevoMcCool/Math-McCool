Definition Of Limits

A limit of f(x) as x approaches a value descibes the behavior of the function around that value.

In other words, a limit represents the y-value based on the function's curve, not the corresponding x-value.

Example of Limits

Let's suppose we define f(x) = (x-2)2 x-2 . If we were to try and calculate f(2) we would get an error.

This is because the denominator of f(2) is 2-2, which is zero. Since we cannot divide by zero, the f(2) is undefined. On the graph, there is a hole at x=2.

However, using any of 3 methods, we can still find lim f(x) x->2

Solving Limits Numerically (GC required)

Let's suppose we have the previous example of f(x) and would like to solve it numerically. This means we are going to use a table of values to determine the limit.

To get an accurate table of values on a graphing calculator, first enter the function in by pressing Y1 and typing the function into your calculator. Remember: Calculators follow pemdas, so parenthesis are important. Your function entry should look like Y1=(x-2)2/(x-2) .

Once done, you need to access the calculator's table settings. To do this, press 2nd + window (located next to the Y1 button). Leave the bottom two options on Auto

After that, change TblStart to the value that X is approaching. In this case, set TblStart=2.Change the second option (Delta Table) to a relatively small number, <=0.01.


Once you have your table set up, click 2nd + graph. You should see a table similar to the table on the right. Scroll upwards until 2 is in the middle.

As you can see, as the x-values get closer to 2, the y-values get closer to 0.

This means that we can conclude that lim f(x) = 0 x->2

Remember, the limit is determined by the behavoir around the point. So even though the point is undefined, there is still a limit at that point.


In my personal opinion, this is the best way to calculate limits. It is fast, simple, and removes any major chance of human error. However, you won't have a graphing calculator for all parts of the AP exam, so it is important to learn the other ways to find limits.

Solving Limits Graphically

Suppose the graph of g(x) is to the right, and we needed to solve lim g(x) x->4

Since we don't have a function equation, we need to solve this limit graphically. To do this, give your best estimate for the function value of x=4, which is 6. You can use a pencil and a straight-edge to help get your answer.


In my personal opinion, this method should ONLY be used when you are only given a graph. There could be a high level of error with this method and is not recommended.

Solving Limits Algebraically

Let's define f(x) = x2+5x+6 x+3 and attempt to find lim f(x) x->-3

Your first step is to check if direct substiution will work by plugging -3 into the denominator. When you do this, you get -3 + 3 = 0, which is a divide by zero error. Direct substiution won't work yet.

The next step is to check if x-3 is a factor of the numerator. To do this, you can factor the numerator using quadratic factorting, which results in (x+2)(x+3). Your function now looks like f(x) = (x+2)(x+3) x+3

You can then cancel the x+3 in the numerator with the x+3 in the denominator. This looks like lim (x+2)(x+3) x+3 x->-3 = lim x+2 x-> -3

Now you can directly substitute -3 into x+2 --> -3+2 --> -1

lim f(x) = -1 x->-3

Where Limits Dont exist

As much as I love limits, sadly, they don't always have my back. It's kind of an abusive relationship. Anyways, they wont always exist.

They wont exist when the left side of the function and the right side of the function approach different points (see example 1)

The limit wont exist if either side of the function approaches infinity (denote it as infinity, see example 2)

The linit wont exist if the function is oscilatting (going up and down rly fast). You can identify oscilation by checking if the frequency of waves is increasing exponentially. (see example 3)

One Sided Limits

Let's denote f(x) as the function that forms the graph on the right. Let's attempt to solve lim f(x) x->0 graphically.

As you can see, the graph clearly approaches 1 from the right, and -1 from the left. However, as stated above, this means lim f(x) DNE x->0

However, if we wanted to determine the limit from the left, we would denote this as lim f(x) x->0-. Notice the negative sign AFTER the zero. This only accounts for all the points left of zero, so the answer is -1. To repeat the process from the right, we would do lim f(x) = +1 x->0+

To solve this numerically, you can use a table similar to the one on the right, but only focus on the part of the table you need (for left-hand limits, use the top half of the table. For right-hand limits, use the bottom half).

Types of Discontinuities

A continuous function is a function with no jumps, holes, or vertical asymtopes*. For example, y= ex is continuous, buy y= 1/x is not. You can prove continuity of f(x), x=c by stating lim f(x) x->c- = lim f(x) = f(c) x->c+

A removable discontinuity looks like a hole in the graph. Essentially, if you just shaded over that one point, the graph would be continuous. You can prove a removable discontinuity by stating lim f(x) x->c- = lim f(x) x->c+ , and lim f(x) x->c- != f(c)

A jump discontinuity looks like a gap between 2 points (refer to this function). You can prove a jump discontinuity by stating the left and right handed limits arent equal, but both limits exist.

You can prove an INFINITE DISCONTINUITY by proving at least one of the one-sided limits goes to infinity. This is also called a VERTICAL ASYMPTOPE

You won't recieve an oscilatting discontinuity, but in most cases, it follows the behavior of a removable discontinuity inside of a wavy function.

Properties Of Limits

The properities of limits say that, for any limit, you can separate any operation and take the limit of both parts. For example:

lim f(x)*g(x) x->c = lim f(x) x->c * lim g(x) x->c ||     ||lim f(x) - g(x) x->c = lim f(x) x->c - lim g(x) x->c

These rules apply for all operations, not just addition and subtraction. We can use these rules to solve limits of special values.

Limit to Zero with Sine

lim sin(cx) cx = 1 x->0

Note that for this to work, the c in the numerator must equal the c in the denominator. Here is an example:

lim sin(6x) 6x = 1 x->0   lim sin(4x) 3x = ?? x->0

The second limit !=1 because 4x != 3x

Limit to Zero with Cosine

lim cos(cx) ax DNE x->0

For cosine, the c in the numerator does not have to equal the c in the denominator. Here is an example:

lim cos(5x) 5x DNE x->0   lim cos(7x) 10x = DNE x->0

Both limits don't exist.
