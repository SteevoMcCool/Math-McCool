Definition Of Limits

A limit of f(x) as x approaches a value descibes the behavior of the function around that value.

In other words, a limit represents the y-value based on the function's curve, not the corresponding x-value.

Example of Limits

Let's suppose we define f(x) = (x-2)2 x-2 . If we were to try and calculate f(2) we would get an error.

This is because the denominator of f(2) is 2-2, which is zero. Since we cannot divide by zero, the f(2) is undefined. On the graph, there is a hole at x=2.

However, using any of 3 methods, we can still find lim f(x) x->2

Solving Limits Numerically (GC required)

Let's suppose we have the previous example of f(x) and would like to solve it numerically. This means we are going to use a table of values to determine the limit.

To get an accurate table of values on a graphing calculator, first enter the function in by pressing Y1 and typing the function into your calculator. Remember: Calculators follow pemdas, so parenthesis are important. Your function entry should look like Y1=(x-2)2/(x-2) .

Once done, you need to access the calculator's table settings. To do this, press 2nd + window (located next to the Y1 button). Leave the bottom two options on Auto

After that, change TblStart to the value that X is approaching. In this case, set TblStart=2.Change the second option (Delta Table) to a relatively small number, <=0.01.


Once you have your table set up, click 2nd + graph. You should see a table similar to the table on the right. Scroll upwards until 2 is in the middle.

As you can see, as the x-values get closer to 2, the y-values get closer to 0.

This means that we can conclude that lim f(x) = 0 x->2

Remember, the limit is determined by the behavoir around the point. So even though the point is undefined, there is still a limit at that point.


In my personal opinion, this is the best way to calculate limits. It is fast, simple, and removes any major chance of human error. However, you won't have a graphing calculator for all parts of the AP exam, so it is important to learn the other ways to find limits.

Solving Limits Graphically

Suppose the graph of g(x) is to the right, and we needed to solve lim g(x) x->4

Since we don't have a function equation, we need to solve this limit graphically. To do this, give your best estimate for the function value of x=4, which is 6. You can use a pencil and a straight-edge to help get your answer.


In my personal opinion, this method should ONLY be used when you are only given a graph. There could be a high level of error with this method and is not recommended.
